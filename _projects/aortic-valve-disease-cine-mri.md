## Overview

This project focused on detecting aortic valve pathology from 3-chamber cine cardiac MRI using deep learning-based image classification.

## Problem

Aortic valve disease assessment typically depends on expert interpretation and multimodal clinical evaluation. Automated analysis from cine MRI has potential to support scalable screening, prioritisation, and quantitative research workflows.

## Approach

I developed classification workflows for 3-chamber cine MRI, including data preparation, model training, evaluation, and interpretation of classification performance.

## Technical contributions

- Built deep learning classification workflows for cardiac cine MRI.
- Compared model performance across convolutional neural network backbones.
- Evaluated classification performance using clinically relevant metrics.
- Considered model interpretability and practical translation in scanner-based workflows.
- Contributed to a project that supported progression towards a scanner-based trial phase.

## Tools and methods

Python, PyTorch, MONAI, CNNs, DenseNet, ResNet, cardiac cine MRI, classification metrics, ROC analysis, Grad-CAM.

## Relevance

This project demonstrates applied computer vision experience in a clinically relevant setting, with emphasis on reproducible model development, evaluation, and translational potential.
