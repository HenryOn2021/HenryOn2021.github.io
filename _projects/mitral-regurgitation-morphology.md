## Overview

This project used clinically interpretable mitral annular morphology features extracted from cine cardiac MRI to classify mitral regurgitation.

## Problem

Deep learning models can be effective but may be difficult to interpret clinically. For mitral regurgitation, morphology-based features provide a transparent alternative by linking predictions to measurable cardiac structure and motion.

## Approach

I led the project end-to-end, including feature preparation, model development, validation, and result interpretation. The project used engineered morphological features and classical machine learning models to classify disease status.

## Technical contributions

- Designed a structured machine learning workflow for feature-based classification.
- Performed feature engineering and model comparison.
- Evaluated performance using standard classification metrics.
- Interpreted results in the context of clinically meaningful cardiac morphology.
- Documented the workflow for reproducibility and portfolio use.

## Tools and methods

Python, NumPy, pandas, scikit-learn, Matplotlib, feature engineering, cross-validation, classification metrics.
