## Overview

This project focused on multi-class bi-atrial segmentation from LGE-MRI in a challenge setting, requiring robust performance under limited data and runtime constraints.

## Problem

Accurate atrial segmentation from LGE-MRI is technically challenging due to complex anatomy, variable image quality, and limited annotated datasets.

## Approach

The pipeline combined region-of-interest extraction, 3D segmentation, targeted preprocessing, post-processing, and structured evaluation.

## Technical contributions

- Contributed to a challenge-oriented segmentation workflow.
- Used a pragmatic multi-stage approach combining ROI localisation and segmentation.
- Applied preprocessing and post-processing to improve robustness.
- Evaluated segmentation performance using Dice and HD95.
- Balanced model performance with runtime and challenge constraints.

## Tools and methods

Python, nnU-Net, medical image preprocessing, NumPy, SimpleITK, Dice, HD95, connected-component post-processing.
